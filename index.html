Quiz Answers

What is semantic HTML?

Semantic HTML is a way of writing HTML which describes to both the browswer and the user what is happening in the HTML. It allows you to see what would be in the content based on the HTML element being used.

How do you import a stylesheet into a HTML document?

You have a few options when importing a stylesheet into a HTML document. Three are:

  1. Inline - by using the style attribute inside HTML elements
  2. Internal - by using a <style> element in the <head> section
  3. External - by using a <link> element to link to an external CSS file. This is done with the format:
    <link rel="stylesheet" href="./cssfolder/.css">

How do you import javascript in to a HTML document?

You can include javascript in your HTML document in a couple of ways:

  1. Embedding the code in your HTML document using <script> tags
  2. External - by using a <script> tag in your document to link to an external js file
    script type="text/javascript" src="./js/jsfilename.js">

What meta tag can you use to fix the mobile screen width?

Viewport is the meta tag that can be used. The viewport is the user's visible area of a web page and varies with different screen sizes. As with this current page the viewport can be set by:
<meta name="viewport" content="width=device-width, initial-scale=1">

How do you apply CSS only to certain screen sizes?

To apply CSS for only certain screen sizes you can use @media rules. You can check many things using @media queries such as the width and height of viewport and device, orientation, resolution, etc. As with the attached css it could look like:
@media (min-width: 768px) and (max-width: 1023px){}

What does mobile first mean when writing CSS?

Mobile first is designing and writing your CSS for a mobile device first and then customising the design for the larger and arguably easier larger devices afterwards. This way you have an improved experience on mobile devices.

How does a browser decide the order to apply CSS?

CSS is applied using specificity. Certain selectors are scored using specificity if they have multiple declarations and the selector with the scores best using the rules gets the CSS applied. If css specificity is equal then the last selector gets chosen.

Name three javascript best practices.

  1. Use naming conventions for files, variables, functions, etc
  2. Use adequate internal comments to highlight and explain certain areas of code and functions
  3. Make your code modular and only use functions for one task and use functions to avoid repeating code sections

How do you run a javascript function when a button is pressed?

You can use an onClick() event with your button to call your javascript function. It would look like:
<button onclick="callMyFunction()">Go Function Go</button>

What is the difference between == and === in javascript?

== will compare two variables regardless of their datatype
5 == "5" is true
=== will compare two variables but will use a strict check for the datatype as well
5 === "5" is false
